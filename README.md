## Project Overview

**Frank Vision Helmet** is a groundbreaking accessibility device that combines cutting-edge AI, real-time computer vision, and intelligent hardware to provide independence and empowerment to visually impaired individuals. 
 
 
### ✨ What Makes Frank Unique

#### 1. 📞 **Dual-Mode Calling System**
- **SIM Card Support**: Make direct calls using cellular networks
- **Twilio Integration**: Cloud-based alternative calling
- **Voice-Controlled Dialing**: Simply say "Call Mom" or "Call Sriram"
- **Emergency Ready**: Quick access to emergency contacts

#### 2. 👤 **Intelligent Face Recognition**
- **3-Minute Learning**: After 3+ minutes of conversation, Frank learns faces
- **Automatic Identification**: Recognizes and greets people by name next time
- **Local Processing**: All face data stored privately on device (no cloud)
- **Continuous Learning**: Improves recognition accuracy with each interaction

#### 3. ❄️ **Advanced Cooling System**
- **Thermal Efficiency**: Reduces device temperature by 30-40°C
- **Extended Battery Life**: 40% longer usage time (8+ hours)
- **Non-Glass Design**: Uses liquid coolant instead of heavy glass helmet
- **Lightweight**: 200g lighter than traditional glass helmets
- **Cost-Effective**: Passive cooling reduces battery drain
- **Improved Performance**: Maintains optimal Pi Zero W operating temperature

#### 4. 🎯 **Future Enhancement**
- 360 degreee camera for high end accessibility
- optimized battery and storages

## 🎯 Specifications

### Hardware
```
CPU:        ARM Single-core 1 GHz
RAM:        512 MB
GPU:        VideoCore IV
Camera:     8MP (Pi Camera v2)
Storage:    8GB microSD
WiFi:       802.11b/g/n
Power:      5V 2.5A
LEDs:       3 (GPIO controlled)
Weight:     ~300g (with cooling)
```

### Software Requirements
```
OS:         Raspberry Pi OS (Bullseye+)
Python:     3.7+
Dependencies: opencv, face_recognition, gTTS, Twilio
Disk Space: 2.5GB
```

### Performance
```
Vision Processing: <500ms
Face Recognition:  ~200ms
Speech Recognition: Cloud-based
Text-to-Speech:    ~1-2 seconds
Battery Life:      8-10 hours
Operating Temp:    35-45°C
```
